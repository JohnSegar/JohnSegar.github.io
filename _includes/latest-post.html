Spacey

Spacey is a Tetris inspired, one player game. It was the first app I built during the WDI course at GA, and was built using HTML5, CSS, JavaScript & jQuery using OOP.

Weddit

Weddit is an app used for those getting married, where all the information for their wedding can be put in one place. The guests can also comment on, and upload photos to the individual wedding they are attending. Weddit is a RESTful app built with Ruby on Rails.

ZNN

ZNN was a group project, and the third whilst at GA. The idea behind it was to make a website for all information required in case of a hypothetical zombie outbreak. It was built in JavaScript, and incorporated Node, Express and Mongo. A number of API's were used, including the Google Maps and YouTube API.

Let's Play Together

Lets Play Together was my final project at GA. Users can login and select which game they are currently playing (using the IGDB API) and then see all other users who are also playing that same game. Alternatively a user can search all users by their name or what they are playing. It was built using the AngularJS framework.
